Qt Quick (Qt User Interface Creation Kit)

Ключевой особенностью Qt 4.7 и Qt Creator 2.1 является Qt Quick: высокоуровневая UI технология, которая позволяет разработчикам и дизайнерам работать вместе над одним проектом и создавать красивые, анимированные программы. Qt Quick включает:

  1. QML дизайнер для Qt Creator IDE
  2. QML (Qt Meta-Object Language): легко используемый, декларативный язык
  3. QtDeclarative: новый модуль, предоставляющий декларативный подход для разработки динамичных пользовательских интерфейсов

Для работы с QML нет необходимости быть разработчиком. Описать интерфейс и его поведение очень просто не имея навыков программирования на C+. Но при необходимости элементы дизайна можно расширить с помощью C+.

Scope (Qt Creator 2.1 и Quick)

  • Мастер создания и менеджер QML проектов. (новый формат файла для приложений на чистом QML)
  • Расширенный редактор QML с подсветкой синтаксиса, автодополнением кода, встроенной справкой и др.
  • Qt Quick Designer: WYSIWYG редактор для создания Qt Quick пользовательских интерфейсов (technical preview)
  • Qt Quick Components: Основные UI элементы, которые соответствуют идее Look & Feel
