This is a collection of older release plans taken from the history of the Qt Creator Releases site, mainly here for archaeological purpose. Upcoming releases are handled at Qt Creator Releases.

Unless specified the date are the final tentative dates (usually Tuesdays in a week) with actual releases happening on Thursdays.

Qt Creator 3.5:

  • Feature freeze w25 (~Jun 16 2015)
  • Beta Release w28 (~Jul 07 2015)
  • String freeze w40 (~Jul 21 2015)
  • Release Candidate w32 (~Aug 04 2015)
  • Release Final w34 (~Aug 18 2015)

Qt Creator 3.4:

  • Feature freeze w8 (~Feb 17 2015)
  • Beta Release w10 (~Mar 3 2015)
  • String freeze w12 (~Mar 17 2015)
  • Release Candidate w14 (~Mar 31 2015)
  • Easter inbetween
  • Release Final w17 (~Apr 21 2015)

Qt Creator 3.3:

  • Feature freeze w41 (~Oct 15 2014)
  • Beta Release w43 (~Oct 28 2014)
  • String freeze w45 (~Nov 11 2014)
  • Release Candidate w47 (~Nov 25 2014)
  • Release Final w49 (~Dec 9 2014)

Qt Creator 3.2:

  • Beta Release w28 (~Jul 8 2014)
  • String freeze w30 (~Jul 22 2014)
  • Release Candidate w32 (~Aug 5 2014)
  • Release Candidate II or Final Release (~Aug 19 2014)
