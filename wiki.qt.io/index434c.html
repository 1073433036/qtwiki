Instalacja SailfishOS SDK

1. Pobierz instalator SDK

SDK Sailfish'a jest dostarczane w formie graficznego instalatora, który powoduje, że proces instalacji jest naprawdę prosty. Wykorzystanie instalatora jest zalecane. SDK jest dostępne na poniższe platformy:

  • linux 32bit instalator
  • linux 64bit instalator
  • windows instalator(XP/Vista/Windows 7)
  • OS X instalator

Po ściągnieciu odpowiedniego instalatora postępuj zgodnie z dalszymi instrukcjami.

2. Przygotuj środowisko

Usuń wszystkie wcześniejsze zainstalowane wersje SailfishSDK np. wersje alpha.

Jeśli posiadasz zainstalowane SailfishOS Alpha SDK to przed instalacja musisz je usunąć. Uruchom narzędzie "SDKMaintenanceTool", które znajdziesz w foldrze z zainstalowanym SDK ( zazwyczaj w katalogach home/users ), i usuń wszystkie komponenty.

Upewnij się, że spełniasz wszystkie wymagania wstępne

  • Zainstalowany Oracle VirtualBox wersja 4.1.18 lub wyższa.

Jeśli korzystasz z systemu Linux zainstaluj VirtualBox'a dostarczanego przez Twoją dystrybucje. W przypadku innych systemów operacyjnych zalecane jest użycie paczek instalacyjnych bezpośrednio z http://www.virtualbox.org.

  • Około 5GB wolnego miejsca na dysku.
  • Zalecane 4GB pamięci RAM lub więcej.

3 Uruchamianie instalatora

Linux

  1. Przed uruchomieniem instalatora na systemie linux musisz mu nadać uprawnienia do uruchamiania. Aby to zrobić otwórz Terminal i wykonaj następujące polecenie: 
$ chmod +x ~/Downloads/<installer_name><code>
 
# Następnie uruchom instalator jako 'normalny' użytkownik (np. nie jako root). Domyślnie SDK zostanie zainstalowane w SailfishOS/ w katalogu home.
$ ./Downloads/<installer_name></code>

Windows

  1. Zlokalizuj ściągniety instalator przy użyciu eksploratora lub cmd.
  2. Następnie uruchom instalator 
> PATHlt;installer_name>.exe<code>
# W przypadku pojawienia się okna dialogowego z ostrzeżeniem bezpieczeństwa informującym, że wydwaca jest nieznany, pozwól aplikacji się uruchomić.
