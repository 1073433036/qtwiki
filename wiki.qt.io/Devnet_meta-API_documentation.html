Qt Developer Network API documentation

The API is a read-only service returning JSON. You can use it to query your own profile and unread forum posts.

The API requires the client to be authenticated.

Information about current logged-in user

HTTP methods GET
URL /qtapi/1/member/profile

Returns a representation of your own profile.

List unread forum posts

HTTP methods GET
URL /qtapi/1/forums/posts/unread

Returns a JSON representation of your unread forum posts.

Example reply

  1. {
  2.     "forum_posts": [
  3.         {
  4.             "title": "An unread post",
  5.             "url": "http://developer.qt.nokia.com/forums/viewthread/100/",
  6.             "poster": "http://developer.qt.nokia.com/member/50",
  7.             "type": "new_topic"
  8.         },
  9.  
  10.         {
  11.             "title": "An important unread post",
  12.             "url": "http://developer.qt.nokia.com/forums/viewthread/112/",

  14.             "type": "new_topic"
  15.         },
  16.  
  17.         {

  24. }
