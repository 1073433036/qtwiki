Qt Project community events

This gives and overview of all the ongoing and comming Qt Project community events, helping co-ordinate all the Qt relevant Qt community events like code.sprints, conferences, talks and so on. Please add your event with link to this page.

Just now we are making an Qt Project Event Box with rollup, hand outs, booth materials, googies and other must have event equipment. We well add a link to the box content when it's ready. (fixme: we are in the process making the box). Transporting such event box is combersome. Please contakt us at Qt Project marketing e-mail list for use and shipping.

We've also making a guide how to handle questions from the press regarding the Qt Project. Please take a look on how to prepeare (fixme: we got to make this).
