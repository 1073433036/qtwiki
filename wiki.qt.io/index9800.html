Πώς να χρησιμοποιήσετε τον εξομοιωτή για το Meego 1.2 Harmattan

Δοκιμασμένο στα Windows7 64bit, αλλά πρέπει να είναι παρόμοιο και σε άλλες πλατφόρμες.

Προαπαιτούμενα

  • Όταν εγκαταστήσετε το Qt SDK 1.1.2 και τον Qt Creator 2.2.1 μπορείτε να χρησιμοποιήσετε τον εξομοιωτή για το Meego 1.2 Harmattan, ο οποίος εξομοιώνει τη χρήση των νέων Nokia N9 and N950.
  • Ο ευκολότερος τρόπος εγκατάστασης είναι : Qt Creator > help menu > Start updater.
  • Επιλέξτε custom και βεβαιωθείτε ότι επιλέγετε experimental > harmattan.

Qt Creator

Επιλογές

Qt Creator > menu options > Maemo > tab Qemu settings:

  • Επιλέξτε hardware acceleration. Software acceleration είναι πολύ αργή και για το λόγο αυτό είναι καλύτερο να κάνετε χειροκίνητες ρυθμίσεις από το να επιτρέψετε αυτόματο έλεγχο. Εάν το hardware acceleration είναι επίσης πολύ αργό τότε το μηχάνημα στο οποίο τρέχετε τον Qt Creator δεν είναι πολύ γρήγορο ή η openGL δεν είναι (σωστά) ρυθμισμένη.

Project Wizard

Αυτά είναι τα βήματα που απαιτούνται για να ξεκινήσετε και να κάνετε configure ένα Harmattan project στον Qt Creator:

  1. Στον Qt Creator ξεκινήστε ένα νέο Qt Quick project και επιλέξτε Harmattan. (Εάν δεν βλέπετε επιλογή για Harmattan αυτό σημαίνει ότι η εγκατάσταση του Harmattan δεν πέτυχε.)
  2. Ακολουθήστε τον οδηγό, δώστε στο project ένα όνομα.
  3. Όταν φτάσετε στην target selection wizard page, κάντε click στο κάτω βέλος στα δεξιά του Harmattan και θα δείτε τέσσερις debug και release versions. Αποεπιλέξτε αυτές που δεν έχουν Meego 1.2 στο όνομα τους.
  4. Ορίστε το application icon και επιλέξτε version control εάν το επιθυμείτε.
  5. Επιλέξτε done και τα απαραίτητα αρχεία θα δημιουργηθούν.

Ο εξομοιωτής σε δράση

  • Θα δείτε το Harmattan button επάνω από το "Run" βέλος στα αριστερά. Κάντε click στο πράσινο πλήκτρο με τη μορφή βέλους και περιμένετε μέχρι να ολοκληρωθεί η φόρτωση.
  • Όταν δείτε την πρώτη home screen με τα app buttons, μπορείτε να κάνετε την swipe gesture (με τον δείκτη του ποντικιού) για να μεταβείτε στη δεύτερη home screen όπου βλέπετε το κείμενο : "Nothing opened yet"
  • Ένα ακόμα swipe σας φέρνει στην τρίτη home screen (notifications). Κάντε swipe ξανά και θα δείτε και πάλι την αρχική home screen.

Υπάρχουν ορισμένα buttons στο επάνω μέρος τα οποία είναι προφανή στη χρήση τους.

Αλλά πού είναι η εφαρμογή μου;

Απλά πατήστε το πράσινο πλήκτρο με μορφή βέλους Run ακόμη μία φορά στον Qt Creator και μετά από μερικά δευτερόλεπτα θα δείτε το εικονίδιο και την εφαρμογή σας στο κάτω μέρος (θα χρειαστεί να κάνετε scroll down).

Δουλεύει η εφαρμογή μου;

Κάντε click στο icon και η εφαρμογή σας θα ξεκινήσει. Σε αυτό το wizard example μπορείτε να κάνετε click σε ένα πλήκτρο για να δείξετε κάποιο κείμενο και έχετε ένα μενού, στο οποίο μπορείτε να μεταβείτε χρησιμοποιώντας το πλήκτρο στο δεξί κάτω μέρος όταν είστε σε landscape mode.

Μπορείτε να κάνετε swipe ξανά και θα δείτε την εφαρμογή σας στη δεύτερη home screen. Εάν ξεκινήσατε άλλες εφαρμογές όπως την κονσόλα θα τις δείτε και αυτές εκεί.
