Qt での開発 (概要)

  • Qt Quick: 開発者と UI デザイナの共同作業で、アニメーションする、タッチパネル対応の UI とアプリケーションの作成が可能です。
  • Qt Mobility: Qt アプリの開発者に、標準的な携帯機器向け機能を利用した開発を可能にします。
  • ゲームの開発
  • Qt WebKit: WebKit は HTML を表示するためのオープンソースの web ブラウザエンジンです。

ツールとアドオン (概要)

  • Qt Creator: Qt 開発者のニーズに適合した、クロスプラットフォーム IDE (統合開発環境)
  • Qt Simulator: Nokia 端末で動く Qt アプリケーション用の高速かつ軽量なシミュレータ
  • Qt Visual Studio Add-in: Visual Studio で Qt アプリケーションを開発するためのツール
  • qmake: Qt のプロジェクトから Makefile を作成するためのビルドツール
  • QtCreator と Git: Git リポジトリシステムを Qt Creator から使う
  • 開発ツール、概要
