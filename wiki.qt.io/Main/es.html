Desarrollo en Qt

Información general Desarrollo de Juegos Qt WebKit: Motor de renderizado open source para visualizar contenido HTML. Qt Quick: Permite a desarrolladores y diseñadores de GUI trabajar en conjunto para crear interfaces gráficas animadas y con soporte para pantallas táctiles. Qt Web Runtime : Permite que aplicaciones Web pueden acceder a algunas APIs de Qt. Qt Mobility: Permite a los desarrolladores de Qt incluir funcionalidad estándar usada por dispositivos móviles.
