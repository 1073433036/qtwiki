Herramientas de Desarrollo para Qt

Qt Creator

Qt Creator Un IDE (Entorno de Desarrollo Integrado) multiplataforma que se ajusta a las necesidades de los desarrolladores Qt.

Herramientas de gestión de proyectos y compilaciones.

qmake es una herramienta de compilación que permite simplificar el proceso de compilación de proyectos de desarrollo en distintas plataformas. qmake automatiza la generación de Makefiles para que sólo sean necesarias unas pocas líneas de información para crear cada Makefile.

Qt Designer

Qt Designer es un poderosa herramienta multi-plataforma para el diseño de Interfaces gráficas y la construcción de formularios para C+. Permite rápidamente diseñar y crear widgets y diálogos usando los formularios que aparecen en pantalla y usando los mismos widgets que se utilizará en la aplicación. Los formularios creados con Qt Designer son totalmente funcionales y pueden ser previsualizados de antemano, para que pueda asegurarse de que se verán y sentirán exactamente como usted los pensó.

Sistema de Ayuda HTML

Qt Assistant es un lector de documentos configurable y redistribuible, que puede ser fácilmente personalizado y redistribuido, lo que proporciona un sistema integrado de ayuda sensible al contexto para su aplicación Qt.

Herramientas de Internacionalización

Qt Linguist provee un conjunto de herramientas que aceleran el proceso de traducción e internacionalización de aplicaciones. Qt tiene soporte simultáneo de múltiples idiomas y sistemas de escritura con un único source tree y una única aplicación binaria.

Qt Simulator

Qt Simulator esta disponible como parte del Nokia Qt SDK, el simulador Qt es usado para probar aplicaciones Qt para dispositivos móviles en un entorno similar al usado por el dispositivo de destino.

Herramientas de terceras partes

  • MinGW es un compilador GCC libre para Windows. Se distribuye con Qt Creator para Windows y se utiliza para crear aplicaciones Qt C+.
  • Squish una herramienta desarrollada por Froglogic para ser usada en la automatización de pruebas en aplicaciones Qt.
  • KD SOAP es una herramienta para la creación de aplicaciones cliente que hagan uso de web services.
