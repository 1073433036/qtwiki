Qt Quick (Qt User Interface Creation Kit | Equipo de Creación de Interfaz de Usuario Qt )

La característica clave de las versiones Qt 4.7 y Qt Creator 2.1 es Qt Quick: una tecnología de Interfaz de Usuario (UI) de alto nivel que permite a los desarrolladores y diseñadores de interfaz de usuario trabajar juntos para crear aplicaciones ligeras animadas e interfaces de usuario táctil. Incluye:

  1. Nuevas herramientas en el IDE de Qt Creator: incluyendo un editor visual que permite a los diseñadores de interfaz de usuario y desarrolladores cooperar, trabajando en el mismo código en un método iterativo
  2. QML (Qt Meta-Object Language | Lenguaje de meta-objetos Qt): un lenguaje declarativo fácil de usar
  3. QtDeclarative?: un nuevo modulo en la biblioteca Qt que permite un nuevo enfoque de programación declarativa

Y aunque no sea necesario tener conocimientos de programación en C++ para usar Qt Quick, es un 100% basado en Qt y puede ser ampliado con C++, limitado solo por su creatividad.

Ámbito (Qt Creator 2.1 y Quick)

  • Asistente de proyectos QML y gestión de proyectos (nuevo formato de archivos para aplicaciones QML puras)
  • Diseñador Qt Quick: editor visual WYSIWYG (lo que ves es lo que obtienes) para crear interfaces de usuario Qt Quick (vista previa técnica)
  • Componentes Qt Quick: Elementos de interfaz de usuario comunes que se ajustan en la plataforma mediante Look&Feel
