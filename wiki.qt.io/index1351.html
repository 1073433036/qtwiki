Qt Creator

Ο Qt Creator είναι ένα διαπλατφορμικό (cross-platform) ολοκληρωμένο περιβάλλον ανάπτυξης [IDE (integrated development environment)], κομμένο και ραμμένο στις ανάγκες των Qt developers.

Αυξημένων δυνατοτήτων επεξεργαστής κώδικα

Ο Qt Creator επικεντρώνεται στην παροχή δυνατοτήτων οι οποίες βοηθούν τους νέους χρήστες του Qt να ξεκινήσουν το συντομότερο δυνατόν, καθώς και επιταχύνει την παραγωγικότητα των έμπειρων Qt developers.

  • Επεξεργαστής κώδικα με C+, QML και ECMAscript υποστήριξη
  • Rapid code navigation εργαλεία
  • Syntax highlighting και αυτόματη συμπλήρωση κώδικα
  • Στατικός έλεγχος κώδικα και style hints καθώς πληκτρολογείτε
  • Υποστήριξη για source code refactoring
  • Context sensitive βοήθεια
  • Code folding
  • Καταστάσεις αντιστοίχισης παρενθέσεων και επιλογής παρενθέσεων

Οπτικός αποσφαλματωτής (Visual debugger)

Ο οπτικός αποσφαλματωτής (visual debugger) για C+ γνωρίζει τη δομή πολλών κλάσεων του Qt, και κατά συνέπεια αυξάνει τη δυνατότητα καθαρής απεικόνισης των δεδομένων του Qt. Επιπρόσθετα, ο Qt Creator εμφανίζει την raw πληροφορία από το GDB με έναν καθαρό και συνεπή τρόπο.

  • Interrupt program execution.
  • Step στο πρόγραμμα γραμμή-γραμμή ή εντολή-εντολή.
  • Ορισμός breakpoints.
  • Διερεύνηση περιεχομένων call stack, watchers, και local και global μεταβλητών.

GUI σχεδιαστές

Ολοκληρωμένος GUI layout και forms builder για C++ projects, ο οποίος επιτρέπει rapidly design και build widgets και dialogs χρησιμοποιώντας on-screen φόρμες με τα ίδια widgets που θα χρησιμοποιηθούν στη εφαρμογή. Οι φόρμες είναι πλήρως λειτουργικές, και μπορεί να γίνει προεπισκόπηση και άμεσα για να διασφαλιστεί ότι θα φαίνονται και θα αποδίδουν ακριβώς όπως τις σχεδιάσατε.

Εγκατάσταση του Qt Creator

Εάν ενδιαφέρεστε να χρησιμοποιήσετε τον Qt Creator για development, κατεβάστε την τελευταία stable release or look for Vendor Packages.

Εάν θέλετε να ελέγξετε τα τελευταία features, fetch τα τελευταία source snapshots από το master ή το 2.1 branch. Εναλλακτικά, παρέχονται nightly updated binary snapshot builds.

Εάν θέλετε να συμμετάσχετε στο development του Qt Creator, βρείτε το project στο gitorious.org

Contributing στο Qt Creator

Κυρίως ακολουθούνται οι διαδικασίες και τα guidelines που χρησιμοποιούνται για το ίδιο το Qt τα οποία μπορείτε να βρείτε στο Qt Contribution Guidelines. Παρακαλούμε βρείτε τις λεπτομερείς πληροφορίες εκεί. Εάν έχετε ερωτήσεις, μην διστάσετε να επικοινωνήσετε π.χ. στο "#qt-creator":irc://irc.freenode.org/#qt-creator IRC κανάλι στο Freenode. (Για μια πλήρη λίστα των IRC καναλιών δείτε OnlineCommunities.)

Ερωτήσεις και Ανατροφοδότηση

Εάν έχετε προβλήματα, παρακαλείσθε να ψάξετε στο Frequently Asked Questions πρώτα. Εάν τα προβλήματα παραμένουν, scan το Qt Creator mailing list archive. Εάν αυτό δεν βοηθήσει, δείτε το Bug Tracker για να ελέγξετε εάν πρόκειται για γνωστό πρόβλημα, ή για να δημιουργήσετε ένα bug report. Τέλος, μπορείτε να ρωτήσετε στην Qt Creator mailing list στέλνοντας ένα mail με θέμα "subscribe" στο qt-creator-subscribe@trolltech.com. Μετά το subscription, μπορείτε να κάνετε post το ερώτημα σας. Μπορείτε επίσης να προσπαθήσετε να κάνετε ερωτήσεις στο "#qt-creator":irc://irc.freenode.org/#qt-creator IRC κανάλι στο Freenode όπου θα βρείτε τόσο Qt Creator developers όσο και Qt Creator users.
