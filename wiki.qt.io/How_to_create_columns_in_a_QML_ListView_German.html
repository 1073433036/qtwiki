Wie man Spalten in einem QML ListView erzeugt

Im folgenden wird ein Weg gezeigt, wie man ein ListView mit mehreren Spalten erzeugt, deren Breite abhängig von der Breite der enthaltenen Daten ist.

Normalerweise sieht ein ListView mit Row-Elementen so aus:

ListView of Row elements

Spalten können erzeugt werden, indem man die Breite der Elemente in den Row-Elementen festlegt.

ListView with columns

Wie man die Text-Breite erhält

Zeichensatzmetriken sind in QML nicht verfügbar. Aber Text-Elemente können mit JavaScript dynamisch erzeugt und ihre Breite abgefragt werden. 

 // Create temporary Text element
 var textElement = Qt.createQmlObject('import Qt 4.7; Text { text: "' + text + '"}',
 parent, "calcTextWidth");
 
// Use textElement.width for the width of the text
 
// Dispose of temporary element
 textElement.destroy()

Spalten in einem ListView erzeugen

Die Spaltenbreiten für das Modell werden mit JavaScript berechnet und in einer Map gespeichert, die als Schlüssel die Elementnamen aus dem Modell verwendet. Diese Spaltenbreiten werden dann im Delegate des ListViews zur Formatierung verwendet.

main.qml

Es folgt eine einfache Benutzeroberfläche zur Demonstration. Als Container wird ein Rectangle-Element benutzt, sodass die Hintergrundfarbe des ListViews und die Farbe der Zeilen übereinstimmen. Beachte, wie die columnWidths Property eingesetzt wird, um die berechneten Breiten zu speichern, welche schließlich im Delegate genutzt werden. 

import Qt 4.7
import "ColumnHelper.js" as ColumnHelper
 
Rectangle {
 width: 620
 height: 200
 color: "#bebebe"
 
ListView {
 id: list
 property variant columnWidths: ColumnHelper.calcColumnWidths(model, list)
 anchors.fill: parent
 model: MovieModel { }
 delegate: MovieItem { }
 }
 
}

ColumnHelper.js

Eine einfache JavaScript-Funktion wird benutzt, um pro Spalte die maximale Breite aller dort enthaltenen Text-Elemente zu berechnen. Das Ergebnis dieser Berechnungen wird als ein einziges Object von der Funktion zurückgegeben, welches alle notwendigen Spaltenbreiten, abfragbar mit den Spaltennamen als Schlüssel, enthält. 

var columns = {}
 
function calcColumnWidths(model, parent)
{
 for (var i = 0; i < model.count; +''i)
 {
 var data = model.get(i)
 for (var key in data)
 {
 if (!columns[key]) {
 columns[key] = 0
 }
 
 var textElement = Qt.createQmlObject(
 'import Qt 4.7;'
'' 'Text {'
 + ' text: "' + data[key] + '" '
 + '}',
 parent, "calcColumnWidths")
 
columns[key] = Math.max(textElement.width, columns[key])
 textElement.destroy()
 }
 }
 return columns
}

function altColor(i) {
 var colors = [ "#bebebe", "#b7b7b7" ];
 return colors[i];
