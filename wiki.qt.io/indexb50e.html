Qt Events

Qt Everywhere Calendar (XML - iCal)

Scope:

  • Events organized by the Qt community.
  • Events welcoming Qt speakers, booths, sponsorship…
  • Events whith many Qt attendees where informal meetups could be organized.

This calendar helps us planning the events where the Qt project should focus. If you want to request sponsorship or support the first step is to add below all the details related to the event.

See also:

  • Qt Contributors Summit
  • FOSDEM
  • SCALE

Adding an entry to the calendar

This calendar is maintained by Quim Gil, Chuck Piercey and Carol Chen - more contributors are welcome.

If you just want to add one event please post it to the marketing list, ideally with a link to a wiki page explaining all the Qt related details.

Looking for the right calendar tool

We are looking for a great tool to publish events in a collaborative way. This would include events organized by others, Qt Project events, Qt local groups meetups… Please join the discussion at the marketing list.

Candidates:

  • Google Calendar, since many people use it.
  • Atlassian team calendars to be considered since the Qt Project is using Atlassian tools and this would offer integration of user accounts and other services.
  • Maybe a combination of both is possible? See Using Google Calendar with Google Apps Integration
